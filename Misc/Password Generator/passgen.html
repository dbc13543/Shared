Password generator v1.2

The purpose of this page is to help you maintain unique, strong passwords for sites without having to memorize a huge number of passwords or write them down where they can be stolen. Simply type in a master password that you have committed to memory, a unique site name or identifier, and a string that tells the algorithm how strong to make your password. Then write down the site name and password complexity string anywhere. If they fall into the wrong hands they will be useless without your master password.

Note that this page does not store any information or send any information to a server. Everything takes place locally on your PC using JavaScript.

Note: For password complexity, enter a string with no spaces. Any numeric value will be interpreted as the required password length. Use the following codes to specify password requirements:

m - Use uppercase characters (mix case)
n - Also use numeric characters
p - Also use punctuation characters
M - MUST include mixed case
N - MUST include numeric characters
P - MUST include punctuation characters

Therefore, a password complexity string of "8MNP" will generate an 8-character password with mixed case, and at least one numeric and punctuation character.

If no password complexity string is provided, the algorithm will default to a 6-character password in all lowercase. Minimum password length is 6 characters.
