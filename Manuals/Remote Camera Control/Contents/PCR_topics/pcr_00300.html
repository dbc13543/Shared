Names and functions of the components of the Remote Camera Control window
When you start Remote Camera Control, the [Remote Camera Control] window is displayed. The names and functions of the components of the window are as follows:


1 Display panel Displays the shooting settings information (exposure mode, shutter speed, sensitivity, etc.).
->Names and functions of the components of the display panel
2 Operation panel Contains the buttons used to control shooting.
->Names and functions of the components of the operation panel
3 Setup panel You can change settings for shooting and saving a still image.
->Setup
Names and functions of the components of the operation panel
The operation panel contains the buttons to be used for shooting an image. The names and functions of the components of the window are as follows:

1 AEL (AE Lock) When you click this button, you can fix the exposure when taking a picture. To release the lock, click the button again.
2 AFL (AF Lock) When you click this button, you can fix the focus when taking a picture. To release the lock, click the button again.
3 FE-L (FEL Lock) When you click this button, you can fix the amount of flash luminescence when taking a picture. To release the lock, click the button again.
4 -/+ Increases or decreases (+/-) the value of the setting item selected on the display panel.
5 MOVIE Button This is the camera's MOVIE button. When you click this button, movie shooting starts. To stop recording, click the button again.
6 Shutter Works as a camera’s shutter. When you click this button, a still image is shot and transferred to the PC.
7 Half-press shutter Works as a camera’s shutter when it is pressed halfway down. The focus is tuned automatically. Then, when the shutter is clicked, a still image is taken with the tuned focus.
8 Interval timer shooting When you click this button, the [Interval Timer Shooting] dialog box is displayed and the interval shooting mode starts. You can set the start time, capture interval, and number of images to capture. Click [Start] so that shooting is performed automatically at the specified capture interval and times. After the interval timer shooting continues for a specified period of time, it stops automatically.
Note
In the interval shooting mode, the drive mode should be set to [Single-Shot Adv.]. In case a different mode is selected, the drive mode will be automatically changed to [Single-Shot Adv.].
Names and functions of the components of the display panel
The display panel shows the current shooting conditions of the camera. The names and functions of the components of the window are as follows:

1 Exposure correction value Displays the exposure correction value and an icon. You can increase or decrease the value by clicking the +/- buttons. Exposure will be corrected when there is a strong contrast (gap in brightness) between the subject and the background. In manual mode shooting, however, no correction will be performed.
2 Focus display Displays the focus status.
3 Mode dial Displays the current mode dial status with an icon.
4 Shutter speed Displays the specified shutter speed. You can increase or decrease the value by clicking the +/- buttons.
5 Diaphragm value Displays the specified diaphragm value. You can increase or decrease the value by clicking the +/- buttons.
6 AE lock Appears when the exposure is locked.
7 ISO studio tungsten/Multi Frame Noise Reduct. Displays the ISO sensitivity value. The higher the value is, the higher the sensitivity becomes. When AUTO ISO mode is selected, [ISO AUTO] is displayed. You can increase or decrease the value by clicking the +/- buttons. For more information, refer to the camera’s manual.
Multi Frame Noise Reduct. can be selected if your camera is compatible.
8 Battery level Displays the remaining camera battery level as a percentage (%) and an icon.
9 Flash compens.

Displays the compensation value of the flash light amount. You can increase or decrease the value by clicking the +/- buttons.

10 Flash mode Displays the current flash mode setting as an icon. For more information, refer to the camera’s manual.
11 Focus mode Displays the current focus mode as an icon. For more information, refer to the camera’s manual.
12 Metering mode Displays the current metering mode as an icon. For more information, refer to the camera’s manual.
